Principais formatações

Negrito (destaque)

Para destacar um termo numa frase, o colocamos em negrito, usando a tag não-semântica <b>.

Nesta outra frase, usamos a tag semântica <strong> para destacar o termo em negrito.

Itálico (ênfase)

Para dar ênfase a um termo numa frase, o colocamos em itálico, usando a tag não-semântica <i>.

Nesta outra frase, usamos a tag semântica <em> para dar ênfase ao termo em itálico.

Texto marcado

Nesta frase, marcamos este trecho, usando a tag <mark>.

Nesta outra frase, marcamos este outro trecho, com um plano de fundo personalizado, usando um parâmetro de estilo <style="background-color">.

Para conceber este marcador de forma universal em uma mesma página, é recomendado fazê-lo no campo <head>.

Texto grande ou pequeno

Nesta frase, escrevemos um texto com palavras grandes e pequenas, usando as tags <big> e <small>, respectivamente.

Texto deletado

Para desconsiderarmos um trecho, o marcamos como excluído, usando a tag <del>, que mostra o trecho tachado.

Texto inserido

Para indicarmos que um trecho foi adicionado posteriormente, o marcamos como inserido, usando a tag <ins>, que mostra o texto sublinhado.

Além disso, um trecho pode ser sublinhado, usando a tag não-semântica <u>.

Texto sobrescrito

Para inserir um texto sobrescrito, como em uma função exponencial, usamos a tag <sup>. Exemplo: x2.

Texto subscrito

Para inserir um texto subscrito, como em uma fórmula de um elemento químico, usamos a tag <sub>. Exemplo: H2O.
